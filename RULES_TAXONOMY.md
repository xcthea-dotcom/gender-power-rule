# Rules Taxonomy

## 说明
这份 taxonomy 是基于 `rules.js` 中 `R001-R062` 做的系统性交叉归类版。

原则有三个：
- 一条 rule 可以同时属于多个大类
- 大类代表压迫结构，不代表互斥标签
- 只要某条 rule 在语义上稳定参与某种结构，就应当被列入

所以这不是“唯一归属表”，而是“应有交叉归类表”。

## 使用原则

### 1. 先看压迫结构，不先看表面话题
例如：
- “女生别太主动”表面是恋爱话题，但核心结构是对女性主动性的控制
- “婚后该回归家庭”表面是婚姻话题，但核心结构是把女性资源收回家庭

### 2. 允许重复出现
例如：
- `R017 女强人不幸福` 同时属于“能力贬低”“婚恋惩罚”“家庭角色回收”
- `R024 生孩子是责任` 同时属于“婚育规训”“家庭角色期待”“父系秩序”

### 3. 兜底规则单独处理
- `R030 暂不明确` 只属于兜底类，不参与其他归类
